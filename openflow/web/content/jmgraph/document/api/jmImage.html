Class jmImage

图片控件，继承自jmControl params参数中image为指定的图片源地址或图片img对象， postion=当前控件的位置，width=其宽度，height=高度，sposition=从当前图片中展示的位置，swidth=从图片中截取的宽度,sheight=从图片中截取的高度。
